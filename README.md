# Fitlife Gym 

A modern and responsive website for **Fitlife Gym**, built with **React.js** and **Tailwind CSS**.

## Features
- Responsive design
- Interactive contact form
- Smooth animations with Framer Motion
- Clean and minimalistic UI

## Technologies Used
- **React.js**
- **Tailwind CSS**
- **Framer Motion**
- **React Icons**
